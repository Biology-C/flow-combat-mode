# Flow Combat Mode — 心流打字體驗

Flow Combat Mode 是一個**追求極致專注**的遊戲化打字訓練工具。  
透過高回饋感、低延遲的視覺與音效特效，把每一次敲擊都轉化為心流體驗。

## 線上體驗

👉 **[點我立即體驗 Flow Combat Mode](https://biology-c.github.io/flow-combat-mode/)**

---

## 四大模式

| 模式 | 說明 |
|------|------|
| **FREE** | 自由打字。每個字觸發粒子特效，連打累積 Combo，顏色升階：藍 → 綠 → 金。支援中文輸入法。 |
| **TEST** | 對照目標句子逐字比對。正確字顯示綠色，錯字紅色，全部正確自動進入下一題。 |
| **ZEN** | 呼吸引導 → 思緒清空 → 深度寫作。內建番茄鐘、行為分析報告、加密存儲。 |
| **MD** | 22 關 Markdown 語法練習，邊打字邊即時預覽渲染結果，支援繁體中文輸入。 |

---

## MD 語法練習關卡（22 關）

| 類別 | 關卡 |
|------|------|
| 標題 | 一級 `#`、二級 `##`、三級 `###` |
| 強調 | 粗體 `**`、斜體 `*`、粗斜體 `***`、刪除線 `~~` |
| 列表 | 無序 `-`、有序 `1.`、巢狀 `  -`、待辦未完成 `- [ ]`、待辦已完成 `- [x]` |
| 結構 | 引用 `>`、水平分隔線 `---` |
| 程式碼 | 行內 `` ` ``、程式碼區塊 ` ``` ` |
| 連結 | 超連結 `[]()`、圖片 `![]()` |
| 表格 | 標題行 `\| \|`、分隔行 `\| --- \|` |
| 組合練習 | 標題＋粗體、列表＋程式碼、引用＋斜體 |

## 核心特色

- **零延遲回饋**：Canvas 粒子系統 + Web Audio API，每次擊鍵即時反應
- **Combo 連段**：連續輸入累積連擊，觸發階段性視覺升級（藍／綠／金）
- **畫面震動**：模擬實體鍵盤打擊感
- **IME 支援**：完整支援繁體中文輸入法（注音、倉頡、速成等）
- **游標精準追蹤**：方向鍵移動游標、選取範圍即時反映到視覺顯示
- **即時 Markdown 預覽**：輸入即渲染，支援標題、粗體、表格、程式碼區塊等完整語法
- **ZEN 分析報告**：打字速度折線圖、退格熱點圖、WPM 統計

## 技術棧

- 原生 HTML5 / CSS3 / Vanilla JavaScript（無框架依賴）
- HTML5 Canvas — 粒子渲染引擎
- Web Audio API — 音效合成
- IME Composition Events — 中文輸入法整合

## 本地執行

```bash
git clone https://github.com/Biology-C/flow-combat-mode.git
cd flow-combat-mode
# 用任意靜態伺服器開啟，例如：
npx serve .
# 或 VS Code Live Server
# 或 python -m http.server
```

直接用瀏覽器開啟 `index.html` 亦可（部分音效需要 HTTP 環境）。
