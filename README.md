# Flow Combat Mode (打字戰鬥模式)

Flow Combat Mode 是一個**追求極致專注**的遊戲化純文字編輯器（MVP 版本）。
透過高回饋感、低延遲的視覺與物理特效轉換你的打字輸入體驗，讓你每一次的敲擊都充滿力量並容易進入心流（Flow）狀態。

## 🎬 線上測試連結 (Live Demo)

👉 **[點我立即體驗 Flow Combat Mode](https://biology-c.github.io/flow-combat-mode/)**

*(註：需在 GitHub Pages 啟用後方可正常瀏覽)*

## 🌟 核心特色 (Features)

- ⚡ **超低延遲的即時回饋**：確保運算與視覺特效能在極短時間內完成，保持輸入順滑零卡頓。
- 💥 **動態粒子特效 (Particle System)**：基於 HTML5 Canvas 的高效能粒子爆炸系統，每個字元躍然紙上。
- 🔥 **連擊系統 (Combo Mechanics)**：捕捉你的輸入節奏，連續不間斷的敲擊將觸發階段性的視覺特效升級。
- 📳 **畫面震動 (Screen-shake)**：模擬實體鍵盤的強大打擊感，配合節奏帶來極具沉浸感的視覺震動。
- 🧘 **多種體驗模式**：內建包含一般遊玩模式、極簡專注的 `Zen` 模式以及自由輸入的 `Free` 模式。

## 🛠️ 技術棧 (Tech Stack)

- HTML5 結構
- CSS3 樣式與動畫
- 原生 JavaScript (Vanilla JS) 邏輯與效能優化
- HTML5 Canvas 渲染引擎

## 💻 本地環境測試 (Local Setup)

如果想在本地端運行與修改此專案：
1. 複製本儲存庫到本地：
   ```bash
   git clone https://github.com/Biology-C/flow-combat-mode.git
   ```
2. 進入資料夾，並透過任何簡易伺服器套件開啟（例如 VS Code 的 `Live Server` 擴充功能，或是執行 `python -m http.server`）。
3. 使用瀏覽器開啟 `index.html`。
